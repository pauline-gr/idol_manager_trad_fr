# Traduction française de Idol Manager

Cette traduction n'est pas un mod ! C'est juste une traduction du jeu original. 

![img](img_readme/1.PNG)

## Comment installer la traduction

Afin de jouer à Idol Manager en français, placer ce dossier dans le dossier `Idol Manager/IM_Data/StreamingAssets/Languages/fr` du jeu.

Une fois le dossier placé, vous pourrez sélectionner le français dans les options du jeu.

Si vous trouvez des erreurs ou des fautes de frappe, n'hésitez pas à les signaler en créant une issue ou en proposant une pull request.

HF !
